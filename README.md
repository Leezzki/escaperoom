Bear Rescue: Escape the Year 3 Den

Files:
- index.html — main page (references `style.css` and `script.js`)
- style.css
- script.js

Run locally:
Open `index.html` in a browser (works locally or when deployed to Netlify).

Audio:
This project uses simple WebAudio-generated beeps for correct/wrong feedback. To add real audio files, place `correct.wav` and `wrong.wav` into a `sounds/` folder and the script will attempt to play them if found.
